MDN Defination

Custom properties (sometimes referred to as CSS variables or cascading variables are entities defined by CSS authors that contain specific values to be reused throughout a document. They are set using custom property notation (e.g., --main-color: black;) and are accessed using the var() function (e.g., color: var(--main-color);). Custom properties are case-sensitive and are scoped to the elements they are declared on, as well as their descendants, following the normal rules of CSS inheritance and the cascade.
